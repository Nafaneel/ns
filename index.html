WHAT WAS NEXUSSNARL?

NexusSnarl (AKA "The Bot") was a codename for a project which successfully involved making an application tasked with purposefully jamming a radio network with preprogrammed sounds and other audio contraptions.

HOW DID IT WORK?
HARDWARE

At the heart of the project was a radio specially programmed with the target's frequency and tone which allowed the target to recieve the transmittions.

[??/05/21 - ??/07/21] - During this period a radio was simply connected to the server machine running the code and upon recieving a command it would transmit by VOX (Voice-operated-transmission) which would require audio to be present before the radio would transmit. This had its limitations and would require a burst of a random sound to open up the VOX and allow the sound to be played. The radio feed was then fed into a tiny antenna stuck in the guttering which would barely get a signal into the school (Specifically only inclusion and reception). This was enough to get a few very distored sounds heard inn the school.

[??/08/21 - 12/05/23] - During the summer holidays of 2021, The project got a massive upgrade with the main feature being an aerial pole with a high gain antenna on top of it was put on the rear of the house and fed into my bedroom. This meant the signal could be well recieved by anyone in the school and beyond 😬. The other main hardware feature was a homemade serial interface device which allowed for connection through USB to the software and could control the radio remotely (Triggering transmit).
